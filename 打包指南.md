# GPU监控应用打包指南

## 📦 打包方案概述

本指南介绍如何将GPU监控脚本打包成可执行文件，支持在不同操作系统上直接运行。

使用工具：**PyInstaller** - 将Python应用转换为独立可执行文件

## 💻 系统要求

### Windows 10/11
- Python 3.6+
- pip 包管理器

### Ubuntu 24.04.4 LTS
- Python 3.10+
- pip 包管理器
- NVIDIA驱动（用于GPU监控）

## 🔨 打包步骤

### 方案A：在Windows上打包

#### 1. 打开命令行
在gpu_win文件夹内，右键选择"在此处打开命令窗口"或PowerShell

#### 2. 安装PyInstaller
```bash
pip install pyinstaller
```

#### 3. 运行打包脚本
```bash
build_windows.bat
```

或手动执行：
```bash
pyinstaller build_executable.spec
```

#### 4. 获取可执行文件
打包完成后，可执行文件位于：
```
dist/gpu_monitor.exe
```

### 方案B：在Ubuntu上打包（推荐用于Ubuntu部署）

#### 前置条件
首先安装必要的系统依赖：
```bash
# 更新系统
sudo apt update

# 安装Python和pip
sudo apt install python3 python3-pip python3-dev

# 安装NVIDIA驱动和nvidia-utils
sudo apt install nvidia-utils

# 安装编译工具（PyInstaller需要）
sudo apt install build-essential
```

#### 打包步骤

##### 1. 进入项目目录
```bash
cd /path/to/gpu_win
```

##### 2. 安装依赖
```bash
pip3 install -r requirements.txt
pip3 install pyinstaller
```

##### 3. 运行打包脚本
```bash
chmod +x build_linux.sh
./build_linux.sh
```

或手动打包：
```bash
python3 -m PyInstaller \
    --onefile \
    --console \
    --add-data "templates:templates" \
    --hidden-import=flask \
    --name gpu_monitor \
    app.py
```

#### 4. 获取可执行文件
打包完成后，可执行文件位于：
```bash
dist/gpu_monitor
```

## 🚀 运行可执行文件

### Windows
```bash
dist\gpu_monitor.exe
```

### Ubuntu/Linux
```bash
chmod +x dist/gpu_monitor
./dist/gpu_monitor
```

或直接：
```bash
dist/gpu_monitor
```

## 📋 打包文件说明

| 文件 | 用途 |
|------|------|
| `build_windows.bat` | Windows自动打包脚本 |

## ⚙️ 高级配置

### 自定义PyInstaller参数

编辑 `build_executable.spec` 文件：

```python
# 例如，修改输出文件名
exe = EXE(
    ...
    name='my_gpu_monitor',  # 改成你想要的名字
    ...
)
```

### 修改监听端口

编辑 `app.py` 最后一行：
```python
# 原来
app.run(debug=False, host="0.0.0.0", port=5000)

# 改成
app.run(debug=False, host="0.0.0.0", port=8080)
```

然后重新打包。

## 🆘 常见问题

### Q1: 打包时出现"找不到Flask"
**A:** 确保已安装Flask：
```bash
pip install flask
```

### Q2: 可执行文件体积很大（>100MB）
**A:** 这是正常的，PyInstaller会打包Python运行时和所有依赖。可以通过以下方式减小：
```bash
pyinstaller --onefile --clean ...  # 添加--clean参数
```

### Q3: Ubuntu上运行时找不到nvidia-smi
**A:** 需要安装NVIDIA驱动：
```bash
sudo apt install nvidia-utils
```

### Q4: "Permission denied" 错误
**A:** 给执行权限：
```bash
chmod +x dist/gpu_monitor
```

### Q5: 打包失败，提示缺少模块
**A:** 可能需要添加hiddenimports，编辑spec文件：
```python
hiddenimports=['flask', 'werkzeug'],
```

## 📦 分发可执行文件

打包完成后，`dist/gpu_monitor` 或 `dist/gpu_monitor.exe` 可以直接分发给其他用户使用，无需安装Python。

### Ubuntu分发

1. 在Ubuntu 24.04.4上打包
2. 分发 `dist/gpu_monitor` 文件
3. 用户只需运行：
   ```bash
   chmod +x gpu_monitor
   ./gpu_monitor
   ```
