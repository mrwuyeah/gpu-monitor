# GPU实时监控 - 详细使用指南

## 📋 概述

这是一个基于Web的GPU监控系统，可以实时显示NVIDIA GPU的运行状态，包括显存占用、使用率、温度、功耗等信息。

## 🚀 快速开始

### 方案一：使用批处理文件（最简单）

1. **双击运行** `启动监控.bat` 文件
2. 看到控制台输出信息后，**打开浏览器**输入：
   ```
   http://localhost:5000
   ```

### 方案二：使用PowerShell

1. **右键** `启动监控.ps1` → 选择 "用PowerShell运行"
2. 如果弹出安全警告，选择 "是"
3. 在浏览器打开：`http://localhost:5000`

### 方案三：手动运行（如需自定义设置）

1. **打开命令行**（cmd或PowerShell）
2. **进入文件夹**：
   ```bash
   cd C:\Users\你的用户名\Desktop\VSCODE\gpu_win
   ```

3. **第一次使用需要安装依赖**（只需一次）：
   ```bash
   pip install -r requirements.txt
   ```
   或直接安装Flask：
   ```bash
   pip install flask
   ```

4. **启动应用**：
   ```bash
   python app.py
   ```

5. **看到以下输出说明成功启动**：
   ```
   GPU监控服务启动中...
   请在浏览器打开: http://localhost:5000
   ```

## 🌐 打开网页

### 本机访问（推荐）
```
http://localhost:5000
或
http://127.0.0.1:5000
```

### 从其他电脑访问（需要在同一局域网）

1. **查看本机IP地址**（在cmd或PowerShell运行）：
   ```bash
   ipconfig
   ```
   找到 "IPv4 地址"，例如 `192.168.1.100`

2. **在其他电脑的浏览器打开**：
   ```
   http://192.168.1.100:5000
   ```

## 📊 界面说明

网页显示内容：
- 🎯 **GPU索引** - 显卡编号（如GPU0、GPU1等）
- **GPU名称** - 显卡型号
- **GPU计算使用率** - GPU核心占用百分比
- **显存使用率** - 显存占用百分比
- **显存使用量** - 已用/总容量（MB）
- **温度** - GPU芯片温度（°C）
- **功耗** - 当前功率消耗（W）
- **工作状态** - 显示"工作中"或"空闲"

### 状态指示灯
- 🟢 **绿色** - 正常使用（<50%）
- 🟡 **黄色** - 高使用率（50%-80%）
- 🔴 **红色** - 非常高使用率（>80%）

## 🔧 配置修改

### 修改端口（如5000被占用）

编辑 `app.py` 最后一行：
```python
# 原来：
app.run(debug=False, host="0.0.0.0", port=5000)

# 修改为（例如改成8080）：
app.run(debug=False, host="0.0.0.0", port=8080)
```

然后重启应用。

### 只允许本机访问

编辑 `app.py` 最后一行：
```python
# 原来：
app.run(debug=False, host="0.0.0.0", port=5000)

# 修改为：
app.run(debug=False, host="127.0.0.1", port=5000)
```

## ⚠️ 常见问题

### Q1: 显示"无法获取GPU信息"
**A:** 可能原因：
- 没有NVIDIA显卡
- 显卡驱动过期或未安装
- nvidia-smi不在系统PATH中

解决方案：
1. 确认有NVIDIA GPU（检查设备管理器）
2. 更新显卡驱动（NVIDIA官网）
3. 重启电脑

### Q2: 出现"ModuleNotFoundError: No module named 'flask'"
**A:** 需要安装Flask依赖：
```bash
pip install flask
```
或
```bash
pip install -r requirements.txt
```

### Q3: 端口5000已被占用
**A:** 修改配置文件中的端口号（见上面的配置修改章节）

### Q4: 浏览器访问时超时
**A:** 确保：
1. 应用仍在运行（检查命令行窗口）
2. 使用正确的地址和端口
3. 防火墙未阻止（如需要可以关闭防火墙或添加例外）

### Q5: 刷新网页后数据消失
**A:** 这是正常的，后端会实时更新数据，只需等待一秒。

- 可以同时打开多个浏览器窗口查看

- 如遇问题，可以重启应用解决大多数问题
