# GPU 监控应用完整部署指南

本项目只保留 **Python 源码运行** 和 **PyInstaller 打包** 两种方式，不再包含 Docker 部署。

## 1. 直接运行源码

### 安装依赖
```bash
pip install -r requirements.txt
```

### 启动
```bash
python app.py
```

浏览器访问：
- `http://localhost:5000`
- `http://localhost:5000/history`

## 2. Windows 打包

### 一键打包
```bash
build_windows.bat
```

### 输出文件
- `dist\\gpu_monitor.exe`

## 3. Linux 打包

### 一键打包
```bash
chmod +x build_linux.sh
./build_linux.sh
```

### 输出文件
- `dist/gpu_monitor`

## 4. 打包脚本逻辑

### Windows 脚本
- 安装 `requirements.txt`
- 安装 PyInstaller
- 清理旧 `build/` 和 `dist/`
- 打包 `app.py` 和 `templates/`

### Linux 脚本
- 安装 `requirements.txt`
- 安装 PyInstaller
- 清理旧 `build/` 和 `dist/`
- 打包 `app.py` 和 `templates/`

## 5. 依赖说明

`requirements.txt` 中保留运行所需的第三方库，例如：
- Flask
- psutil

PyInstaller 会把 Python 运行时和这些依赖一起打包。

## 6. 常见问题

### 找不到 `nvidia-smi`
请先安装 NVIDIA 驱动，并确认命令行可以执行：
```bash
nvidia-smi
```

### 打包后无法打开页面
确认程序已启动，并访问：
```text
http://localhost:5000
```

### 历史页数据太多
系统会按时间范围均匀采样，图表最多保留 100 条点位，确保趋势可读。
