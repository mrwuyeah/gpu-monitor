# GPU 监控应用完整部署指南（2026/5/22 多VLLM进程监控版本）

本项目提供两种使用方式：
1) 直接运行源码（Python）
2) PyInstaller 打包成可执行文件（Windows/Linux）

## 1. 直接运行源码

### 安装依赖
```bash
pip install -r requirements.txt
```

### 启动
```bash
python app.py
```

浏览器访问：
- `http://localhost:5000`
- 历史页：`http://localhost:5000/history`

## 2. Windows 打包

### 一键打包
```bash
build_windows.bat
```

### 输出文件
- `dist\gpu_monitor.exe`

## 3. Linux 打包

### 一键打包
```bash
chmod +x build_linux.sh
./build_linux.sh
```

### 输出文件
- `dist/gpu_monitor`

## 4. 使用 .spec 打包（可选）

- 打包 Web 监控（包含 templates）：
```bash
pyinstaller build_executable.spec
# 或
pyinstaller gpu_monitor.spec
```

- 打包 monitor_gpu.py（如仍需要该脚本）：
```bash
pyinstaller monitor_gpu.spec
```

## 5. 依赖说明

运行时需要：
- NVIDIA 驱动
- `nvidia-smi` 可执行

Python 依赖见 `requirements.txt`。

## 6. 常见问题

### 找不到 `nvidia-smi`
请先安装 NVIDIA 驱动，并确认命令行可以执行：
```bash
nvidia-smi
```

### 打包后无法打开页面
确认程序已启动，并访问：
```text
http://localhost:5000
```

### 模板/静态资源缺失
确认打包时包含了 `templates/`（Windows 用 `templates;templates`，Linux 用 `templates:templates`）。
