# 快速参考

## 直接运行
```bash
pip install -r requirements.txt
python app.py
```

## Windows 打包
```bash
build_windows.bat
```

## Linux 打包
```bash
chmod +x build_linux.sh
./build_linux.sh
```

## 运行打包结果
```bash
# Windows
dist\gpu_monitor.exe

# Linux
chmod +x dist/gpu_monitor
./dist/gpu_monitor
```

## 访问地址
- 实时页：`http://localhost:5000`
- 历史页：`http://localhost:5000/history`

## 说明
- 只保留 Python / PyInstaller 路线
- 不再使用 Docker 部署
- 历史页默认最近 1 小时
- 24 小时及更长范围做均匀采样，最多 100 条用于图表趋势
